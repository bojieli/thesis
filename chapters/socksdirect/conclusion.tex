\section{本章小结}
\label{socksdirect:sec:conclusion}

\sys {}是Linux兼容的高性能用户空间套接字系统。
为了控制平面的可信，本文设计了一个每主机的监控守护进程;
一个点对点无同步数据平面，完全支持fork和多线程套接字共享;
和一个有效利用共享内存和RDMA的环形缓冲区。
\sys {}实现了接近硬件限制的性能，并提高了实际应用程序的端到端性能。

%\sys is a high performance user-space socket architecture that is compatible with existing applications and preserves isolation among processes, while being scalable to multiple cores and many concurrent connections. We carefully remove all major overheads in Linux socket.%\sys can significant reduce communication overhead, especially in a service oriented distributed system environment.
